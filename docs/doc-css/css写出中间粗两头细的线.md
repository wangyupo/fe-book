# css写出中间粗两头细的线

关键字：渐变

```css
background: -webkit-linear-gradient(left, #fff 0%,#a0a0a0 50%,#fff 100%);
```

示例：

<p style="width: 100px;height: 1px;background: -webkit-linear-gradient(left, #fff 0%,#a0a0a0 50%,#fff 100%);"></p>
