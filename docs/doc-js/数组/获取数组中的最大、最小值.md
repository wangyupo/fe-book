# 获取数组中的最大、最小值

Math.max()

> 函数返回一组数中的最大值。

```js
const arrayMax = arr => Math.max(...arr);
// arrayMax([10, 1, 5]) -> 10

或者

const arrayMax = arr => Math.max.apply(null,arr);
// arrayMax([10, 1, 5]) -> 10
```

Math.min()

> 返回零个或更多个数值的最小值。

```js
const arrayMin = arr => Math.min(...arr);
// arrayMin([10, 1, 5]) -> 1

或者

const arrayMax = arr => Math.min.apply(null,arr);
// arrayMax([10, 1, 5]) -> 10
```
