# 两个数组取相同、不同项

> 利用`new Set`生成唯一值的set对象，然后用`filter`过滤即可。

### 相同项

```js
const difference = (arr1, arr2) => {
    let s = new Set(arr2);
    return arr1.filter(x => s.has(x))
}
// difference([1,2,3], [1,2]) -> [3]
```

### 不同项

```js
const intersection = (arr1, arr2) => {
    let s = new Set(arr2);
    return arr1.filter(x => !s.has(x))
}
// intersection([1,2,3], [4,3,2]) -> [2,3]
```
